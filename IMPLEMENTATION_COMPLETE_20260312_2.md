### 구현 내용

#### 1. 월세 입력을 메인 페이지로 이동
**변경 전**:
- NumberPadInput 바텀시트로만 입력 가능
- 클릭 → 바텀시트 → 입력 (2단계)

**변경 후**:
- 매수/전세와 동일한 인라인 슬라이더
- 메인 페이지에서 즉시 조정 가능
- 보증금, 월세 모두 슬라이더 + 프리셋 버튼

#### 2. 모든 슬라이더에 -/+ 버튼 추가
**적용 대상**:
- ✅ 월 저축 가능 금액
- ✅ 현재 보유 자산
- ✅ 매수가격
- ✅ 전세보증금
- ✅ 월세 보증금
- ✅ 월세

**버튼 기능**:
- 슬라이더 양쪽 끝에 8×8 크기 버튼
- 최소 단위(스텝)로 클릭 조정
- Math.max/Math.min으로 범위 제한

#### 3. 월 저축액 슬라이더 스텝 변경
**변경 전**: 50만원 스텝
**변경 후**: 10만원 스텝 (5배 더 세밀한 조정)

| 보유 자산 | 10,000,000 | 0 | 2,000,000,000 | ±10,000,000 |

| 전세 보증금 | 10,000,000 | 50,000,000 | 2,000,000,000 | ±10,000,000 |
| 월세 보증금 | 5,000,000 | 0 | 200,000,000 | ±5,000,000 |
| 월세 | 100,000 | 0 | 3,000,000 | ±100,000 |
